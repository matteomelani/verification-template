# Sessions folder

Optional. A rolling log of LLM session summaries — useful as institutional memory for future LLMs and humans reading the project later.

## Whether to use this

It's optional. Some projects benefit from a narrative log of decisions and progress; others find it noisy. Start with it enabled and disable it if it becomes overhead.

If you decide not to use it, you can delete this folder. The methodology does not require it.

## Format

One file per session, named `YYYY-MM-DD-slug.md`. See `_meta/SCHEMA.md` for the structure.

## Empty for now

This folder is empty in the template. Sessions accumulate as the project develops.
